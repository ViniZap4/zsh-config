# zsh-config

My zsh configuration — cross-platform `.zshrc` with Oh My Zsh, fzf, zoxide, bat, eza, and aliases.

## What's included

- Oh My Zsh with custom `vini4` theme
- Lazy-loaded NVM
- fzf integration with bat preview
- Zoxide smart directory navigation
- eza aliases for modern `ls`
- Cross-platform zsh-autosuggestions (brew / git clone)
- Local overrides via `~/.zshrc.local`

## Install

```bash
./install.sh
```
